## NVM

 NVM（Node version manager）顾名思义，就是Node.js的版本管理软件，可以轻松的在Node.js各个版本间切换，项目源码GitHub

## 下载安装

1. 下载并安装NVM脚本
```bash
curl https://raw.githubusercontent.com/creationix/nvm/v0.13.1/install.sh | bash

source ~/.bash_profile
```

2. 列出所需要的版本
```bash
nvm list-remote
```
